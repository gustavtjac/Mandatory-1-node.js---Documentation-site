# Deployment on Vercel

There are multiple ways of deploying Node.js applications.
[Vercel](https://vercel.com/) has probably one of the most intuitive ways to deploy simple node.js applications.

## How to Deploy

1. Log in with your GitHub account.
2. Click **Add New Project**.
3. Choose the repository that contains your Node.js project.
4. Set environmental variables if needed.
5. Click **Deploy**.
6. Your site is now deployed and will automatically update when you push to your default branch.

### How to Deploy

Sometimes serving your static / public content can mess up with Vercel.

To fix that just add the following .json file to your project root.

```json
{
    "version": 2,
    "builds": [
        {
            "src": "app.js",
            "use": "@vercel/node",
            "config": {
                "includeFiles": ["public/**"]
            }
        }
    ],
    "routes": [
        {
            "src": "/(.*)",
            "dest": "app.js"
        }
    ]
}
```

Switch includeFiles to whatever your public folder is called.
dest should also be set to whatever your project entrance is
