# Deployment on Vercel

There are multiple ways of deploying Node.js applications. 
[Vercel](https://vercel.com/) has probably one of the most intuitive ways to deploy simple node.js applications.

## How to Deploy
1. Log in with your GitHub account.
2. Click **Add New Project**.
3. Choose the repository that contains your Node.js project.
4. Set environmental variables if needed.
5. Click **Deploy**.
6. Your site is now deployed and will automatically update when you push to your default branch.
