# JavaScript callback functions

In JavaScript a callback function is simply put just a function that is passed as an argument to another function which is then called inside that function.

```javascript
function greet(name, callback) {
    console.log('Hello ' + name);
    callback();
}

function sayBye() {
    console.log('Goodbye!');
}

greet('Gustavo', sayBye);

//__CONSOLE OUTPUT__
// Hello Gustavo
// Goodbye!
```

in the example above we pass the reference of the function sayBye() into the parameters of greet() and then call sayBye() inside of our greet function

## When do we use callback functions ?

Without thinking about it you probably use callback functions all the time.

```javascript
button.addEventListener('click', function () {
    console.log('button clicked');
});
```

Above is an example of how many people would create an event listener. The second parameter is also a callback function, its just getting defined directly in the parameters
