# Nodemon

## What is Nodemon
Nodemon is a tool for Node.js developers that automatically reruns your application each time you make a change in the code

So instead of typing this each time you want to start your application 

```bash
node app.js
```

You can just type the following once to activate nodemon

```bash
nodemon app.js
```
Now your application will rerun itself each time you update your code

Nodemon is meant for development only and should not be run in production

## How to install Nodemon

Type the following into your terminal (Make sure you have Node.js installed)

```bash
npm install -g nodemon
```

After this you will be able to use nodemon on all of your node.js projects
