# Spread operator

In JavaScript there's a spread operator represented by three dots `...`

```javascript
...object
```

The spread operator takes all values from inside an object or array and spreads them out.

```javascript
const numbers = [1, 2, 3];

console.log(...numbers); // 1 2 3
console.log(numbers); // [1, 2, 3]
```

It's often used to merge arrays or objects together.

```javascript
const a = [1, 2, 3];
const b = [4, 5, 6];

const merged = [...a, ...b]; // [1, 2, 3, 4, 5, 6]
```

```javascript
const user = { name: 'Gustavo', hobby: 'Health' };
const extra = { age: 22 };

const merged = { ...user, ...extra }; // { name: "Gustavo", hobby: "Health", age: 22 }
```
