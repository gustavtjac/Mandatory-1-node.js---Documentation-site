# Incrementing with ++ & decrementing with --

In JavaScript you can use the ++ and -- operators to either increment or decrement a number by 1.
```javascript
count += 1;
count = count + 1;
count++;

count -= 1;
count = count - 1;
count--;
```
All of the examples above accomplish the same thing, just written in different ways. 

However there's a difference on if you use ++ & -- as prefixes or as suffixes.
```javascript
let i = 5;

console.log(i++);  // prints 5  → i is now 6

console.log(++i);  // prints 7  → i is now 7
```

Using ++ as a suffix will first return and then increment after that while using ++ as a prefix will increment first and then return.
