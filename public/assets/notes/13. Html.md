# HTML - Hyper Text Markup Language

HTML is a language that is used to define content on the web. It's the most used Markup language and all browsers use it.

It's what you're looking at right now!

a HTML site is made up of different tags

```html
<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <link rel="shortcut icon" href="assets/images/logo.png" type="image/png" />
        <title>NoteNode.js | frontpage</title>
        <link rel="stylesheet" href="assets/css/main.css" />
        <link
            rel="stylesheet"
            href="https://cdnjs.cloudflare.com/ajax/libs/highlight.js/11.11.1/styles/atom-one-dark.min.css"
        />
        <script src="https://cdnjs.cloudflare.com/ajax/libs/highlight.js/11.11.1/highlight.min.js"></script>
    </head>

    <body>
        <!-- Header for my documentation site -->
        <header>
            <a href="/"> NoteNode<span>.js</span> </a>
            <input type="search" placeholder="Search..." />
            <nav>
                <a href="/about">About</a>
                <button id="theme-toggle">🌙</button>
            </nav>
        </header>

        <div class="flex">
            <!-- documentation navigation bar  -->
            <aside>
                <h2>Notes</h2>
                <nav id="side-nav-bar"></nav>
            </aside>
            <!-- Main content on frontpage  -->
            <main id="main-section" class=""></main>
        </div>

        <script src="assets/js/header.js"></script>
        <script type="module" src="assets/js/side_bar.js"></script>
        <script type="module" src="assets/js/frontpage.js"></script>
    </body>
</html>
```

The example above is the site that you're looking at right now - some content is just not loaded in at this point.

## What do different tags do?

HTML has a lot of built-in tags that do various different things.

Script tag is used to load in a javascript file

```html
<script src="script.js"></script>
```

Style tag is used to define styling inside of the HTML

```html
<style>
    body {
        margin: 0;
    }
</style>
```

Link tag can be used to import an external stylesheet. It can also be used to set the site icon and various other things.

```html
<link rel="stylesheet" href="assets/css/main.css" />

<link rel="shortcut icon" href="assets/images/logo.png" type="image/png" />
```

HTML sites should at least follow the structure of the example under to meet the browsers expectations

```html
<html>
    <head> </head>
    <body></body>
</html>
```

Most built-in tags have either built in styling and/or functionality.

Some examples are:

```html
<input type="text" />
```

Input allows the client to input values

```html
<button>Click me</button>
```

Button can be clicked.

Even though there are built in tags you dont necessarily have to use them.
You can accomplish the same behaviour by creating your own tag and giving it some styling and/or adding some scripts.
They will get rendered in the browser and behave like the built-in tags, but it might not be the wisest thing to do.

## Semantic HTML

Using HTML semantically has a couple of different pros

- Accessibility - Vision impaired users can better navigate your site
- Better SEO - Google will understand your site far better if you use semantic HTML, and may even rank you higher on searches
- Easier to maintain code - If all tags inside your HTML are <div>'s its gonna be much harder for you or your colleague to understand the code 6 months later
