# Node Package Mananger - NPM

Node.js has a built in package mananger called NPM. NPM is what allows us to install dependencies for our Node.js projects.

to get started with NPM either create a package.json in the root of your project or use the following command

```console
npm init
```

After running the command NPM will ask you a bunch of question to fill out information in the package.json that is getting generated

## package.json

package.json is the file that describes your projects details.

It contains meta-data like the projects name, description and version.
More importantly it also contains the list of dependencies used for the project + our defined scripts.

Under here is an example of a package.json (From this project)

```json
{
    "name": "mandatory-1-node.js---documentation-site",
    "version": "1.0.0",
    "description": "",
    "main": "app.js",
    "type": "module",
    "scripts": {
        "start-dev": "cross-env PORT=8080 NODE_ENV=DEV node app.js",
        "start-prod": "cross-env PORT=80 node app.js",
        "tw": "npx tailwindcss -i input.css -o public/assets/css/main.css --watch"
    },
    "repository": {
        "type": "git",
        "url": "git+https://github.com/gustavtjac/Mandatory-1-node.js---Documentation-site.git"
    },
    "author": "",
    "license": "ISC",
    "bugs": {
        "url": "https://github.com/gustavtjac/Mandatory-1-node.js---Documentation-site/issues"
    },
    "homepage": "https://github.com/gustavtjac/Mandatory-1-node.js---Documentation-site#readme",
    "dependencies": {
        "cross-env": "^10.1.0",
        "express": "^5.2.1",
        "marked": "^17.0.4"
    },
    "devDependencies": {
        "@eslint/js": "^10.0.1",
        "@tailwindcss/cli": "^4.2.1",
        "eslint": "^10.0.2",
        "globals": "^17.4.0",
        "tailwindcss": "^4.2.1"
    }
}
```

## installing dependencies

To install the dependencies described in package.json, use the following command

```console
npm install
```

This will install all of the dependencies and put them into the "node_modules" folder. Its important that we dont push this file to git.

Its also possible to install one dependency at a time using the following command

```console
npm install DEPENDENCY_NAME
```

This will automatically add the dependency to package.json and also install the depedency into node_modules

## running scripts

Inside the script object in package.json it's possible to define scripts.
Usually these scripts get made for starting the application for a certain profile or for testing the code.

You can run a script in the following way:

```console
npm run SCRIPT_NAME
```
