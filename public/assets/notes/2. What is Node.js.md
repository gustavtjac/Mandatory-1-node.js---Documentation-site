# What is Node.js

JavaScript is normally runned in your browser. Node.js allows JavaScript to be compiled on your own machine which expands the possibilities for what you can create with it. This site is an example of a Full-stack Node.js application.

## Running a Node.js application

To run a Node.js application you type the following

```console
node APPLICATION_NAME.js
```

This will run the app right from your machine with no need for a browser.

## Node.js REPL - Read-Eval-Print Loop

Node.js like many other programming languages has a built in REPL.

To acceess the REPL type the following command in your terminal

```console
node
```

This will start the REPL and allow you to write JavaScript code directly in your terminal.
